Two redesign bundles live in `docs/branding/`:
- `pwhl_redesign_bundle_v3_1.zip` — the **canonical / fuller** bundle. README has Parts 1–8 (incl. Part 5 Emotional Design Principles, Part 6 Brand Naming "Northern Ice", Part 7 Competitive Moments, Part 8 Prestige Gradient). Also ships extra files: `EMOTIONAL_UX/EMOTIONAL_UX.md`, `ENHANCEMENTS.md`, `COMPONENTS/MomentumStrip.md`, `TOKENS/prestige-gradient.css`. This is the `Source:` cited by existing RD-001–012 cards.
- `Color scheme redesign request 3.zip` — a **subset**: only `design_handoff_color_system/` (color + emoji + wizard + 2 flagship screens). Its README is byte-identical to v3_1's README **truncated at line 261** — it drops Parts 5–8.

**Why:** "request 3" is the color-system slice handed off separately; v3_1 is the complete package. The new "request 3" zip introduced NO net-new specs vs v3_1 — gaps come from v3_1 Parts 5–8 items that were never carded, not from request 3.

**How to apply:** When asked "what's new in request 3 vs the RD cards," compare against v3_1 (already carded), not just RD-001–012 prose. Real uncovered gaps as of Sprint 22 planning: (1) Brand naming "Northern Ice" decision, (2) team identity colors, (3) trend arrows + upset callout as standalone matchup indicators (Momentum Strip RD-008 only covers the strip), (4) settings editor rule-sheet restructure (RD-007 marks it recolor-only), (5) documented Emotional Design north-star principles artifact.
