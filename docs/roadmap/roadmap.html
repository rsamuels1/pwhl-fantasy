PWHL Fantasy  ·  Product Roadmap  ·  Updated June 12, 2026

The road to a public beta. Seven phases, thirty-two features, one live fantasy season.

The flagship experience is the live fantasy season — drafting real players, setting weekly lineups, and competing in matchups scored from real PWHL games. Here's everything shipped, in progress, and still ahead.

What to build next

Sequenced by token efficiency — each item shows its estimated Claude Pro context cost. Fixed overhead (CLAUDE.md + history) is ~15K per session; the 200K window fits 2–3 quick wins or one heavy lift.
  1. Quick wins — batch 2–3 per session (< 45K tokens each)
  2. Lineup Stats Tab Polish · #28
    Rename "Projected" → "Matchup Proj", hide "This week" between weeks. Single-component edit.
    ~25K
  3. Playoff Experience UX · #30
    Bracket prominence, champion banner, between-round nudge. Polish on existing pages; no schema changes.
    ~40K
  4. Standard — one feature per session (60–90K tokens)
  5. Error Handling · #4
    Empty + loading states across all core pages. Many files, but each change is small and localized. Beta prerequisite.
    ~65K
  6. Weekly Performance Dashboard · #29
    New page replacing the Schedule tab — week-by-week rankings + position-group breakdown. No schema changes.
    ~65K
  7. Mobile Optimization · #3
    Responsive CSS across draft room, matchup, and roster screens. Beta prerequisite.
    ~75K
  8. Team Analysis & Insights · #25
    New Analysis tab — position-group trends, FA recommendations, trade suggestions once #7 lands.
    ~85K
  9. Heavy lifts — plan a fresh session (100K+ tokens)
  10. League Onboarding · #2
    Entirely unbuilt — welcome flow, setup wizard, draft guide. Beta prerequisite despite token cost.
    ~100K
  11. Trade System · #7
    New domain: schema tables, API routes, proposal / review / approval UI. Plan a dedicated session.
    ~130K
  12. Transaction History · #8
    ~45K after #7 — much cheaper once the trade domain exists. Primarily a log view reading the new tables.
    ~45K↓
  13. Waiver Priority + Processing · #5
    Priority ordering, batched claim-resolution jobs, commissioner settings. Schema + service + UI.
    ~110K
Player Legacy (#31) · ~95K: career dashboard, global leaderboard — deferred until one live season completes. Stretch: league-wide storylines (#11 · ~50K) and the rivalry / Hall-of-Fame layer (#17–#18). Replay (Phase 4) stays out of this list.
