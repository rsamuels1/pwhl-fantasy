# Enhancements Added

## Design Principles
PWHL GM should feel:
- Competitive
- Welcoming
- Premium
- Seasonal
- Social

Avoid feeling:
- Enterprise SaaS
- Hockey analytics software
- Generic admin dashboard
- Mobile banking app

## Engagement Suggestions
- Lean into gold as a prestige accent for champions, streaks, and league leaders.
- Add personality to empty states.
- Add matchup momentum chips, trend arrows, and upset indicators.
- Support subtle team identity colors.
- Add a 'Your League at a Glance' summary panel in setup.
