# Founding Commissioner Program

Version: 1.0

Status: Draft

Owner: Product

Purpose: Recruit, support, and learn from an initial cohort of commissioners who will run the first PWHL Fantasy leagues and help validate the MVP.

---

# Program Goals

The Founding Commissioner Program exists to:

1. Validate that leagues can successfully complete a season.
2. Identify usability problems before public launch.
3. Validate league rules and scoring.
4. Generate qualitative feedback from commissioners.
5. Build a small core community around the platform.

---

# Success Criteria

The program is successful if:

- At least 5 leagues complete a draft.
- At least 5 leagues reach Week 4.
- At least 3 leagues complete an entire season.
- Commissioners provide actionable feedback.
- MVP launch blockers are identified before public beta.

# Preferred League Configuration

Default recommendation:

- 8 Teams
- Head-to-Head VP
- 4-Team Playoffs
- Weekly Lineup Lock
- Snake Draft

Reason:

This configuration has the lowest operational complexity and best matches current MVP assumptions.

# Commissioner Profile

Ideal founding commissioners:

- Active PWHL fans
- Existing fantasy sports players
- Community organizers
- Discord moderators
- Reddit community members
- Hockey podcast hosts
- Friends and family willing to actively test

---

# Commissioner Expectations

Commissioners agree to:

## Before Draft

- Create league
- Invite managers
- Schedule draft
- Review league rules

---

## During Draft

- Complete draft
- Report issues
- Capture screenshots if bugs occur

---

## During Season

- Set lineups
- Encourage participation
- Report scoring concerns
- Submit feedback

---

## End of Season

Complete final survey.
