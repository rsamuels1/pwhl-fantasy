# Product Roadmap & Feature Matrix

**Product:** PWHL Fantasy

**Vision:** Create the easiest and most engaging fantasy experience for PWHL fans, including users who have never played fantasy sports before.

**Guiding Principles**

- Simplicity over flexibility
- Opinionated defaults over endless configuration
- Fun weekly engagement over fantasy complexity
- Mobile-friendly experience
- Strong league social experience

---

# Release Phases

## MVP

The minimum feature set required to launch a closed beta and validate product-market fit.

## Launch

Features required for a public launch and a complete season-long fantasy experience.

## Future

High-value enhancements that improve retention, customization, and long-term growth.

# MVP Scope

## Goal

Allow users to complete an entire fantasy season manually with minimal commissioner intervention.

## Required User Journey

### Create League

- Create league
- Invite friends
- Join league

### Draft

- Enter draft room
- Draft players
- Complete roster

### Compete

- Set weekly lineup
- Follow live scoring
- View matchup results
- View standings

### Determine Champion

- VP standings update weekly
- Playoff teams qualify
- Champion crowned

# Features Explicitly Excluded From v1

The following features are intentionally deferred:

- FAAB
- Keeper leagues
- Dynasty leagues
- Custom scoring systems
- Custom roster formats
- Draft pick trading
- Public leagues
- Native mobile apps
- AI features
- Advanced commissioner settings
- Complex playoff structures

These features add complexity without significantly improving the experience for the target audience of casual PWHL fans.

# Success Metrics

## MVP Success

- Users can create leagues successfully.
- Users complete drafts successfully.
- Weekly scoring functions reliably.
- VP standings calculate correctly.
- Playoffs complete successfully.

## Launch Success

- 80%+ of managers set lineups weekly.
- 75%+ of leagues finish the season.
- Less than 5% commissioner intervention required.
- Positive user feedback on ease of use.

## Future Success

- Multi-season retention.
- League renewal year-over-year.
- Increased social engagement.
- Growth beyond early-adopter PWHL fans.
